## 🚀 Running Locally

### 1) Clone

```bash
git clone https://github.com/satyakiabhijit/portfolio-website.git
cd portfolio-website
```

### 2) Install

```bash
npm install
```

### 3) Run

```bash
npm run dev
```

### 4) Build

```bash
npm run build
```
